▼ Os 5 ajustes detalhados

1
Tela de debug não aparece mais pra admin
A caixa amarela do __debugKanban só aparecia quando Kanban ficava vazio E tinha cards no banco. Como vou setar default = "Time" pra admin (e ele vê tudo), Kanban nunca mais ficará vazio pra ele → caixa não dispara. Caso fique vazio em algum cenário extremo, removo a caixa de vez (era pra debug temporário).
2
Toggle só com nome (sem avatar)
Pills horizontais com nome simples. Mais clean, escala melhor com muitos técnicos. Avatares só fazem sentido se a foto realmente identificar a pessoa rapidamente — em tela pequena viram um borrão.
3
Contagem só por RESPONSÁVEL (ignora autor)
Antes contava cards onde resp OU autor === user. Edilson criou uma demanda pro Joacir → tava aparecendo "1" pra Edilson (autor) E pra Joacir (resp). Agora conta SÓ pelo responsável. Edilson não tem mais "1" se ele só foi criador.
4
Default = "Time" pra admin/gerente (forçado)
Já era default na v3.52.4 mas o localStorage antigo (fio_kb_view_admin = "meus") podia sobrescrever. Vou limpar a chave antiga uma vez via flag fio_kb_view_reset_v1 pra garantir que admin abra em "Time".
5
Esconder admin/gerente do toggle (só técnicos aparecem)
Edilson e Jairo são gerentes, não operadores da manutenção. Não faz sentido aparecerem como "técnico filtrável". Filtro novo: só usuários com role producao que tenham acesso ao módulo Manutenção aparecem como pill. Admin e gerente são supervisores — não viram filtro de técnico (só veem via "Time").

Resumo

  • ✓ Toggle só nome (sem avatar) — escala melhor, mais clean
  • ✓ Só técnicos (producao) aparecem — gerentes/admin escondidos
  • ✓ Contagem só por responsável (não autor)
  • ✓ "Time" default pra admin/gerente (limpa cache antigo via flag)
  • ✓ Caixa de debug não dispara mais (admin com Time = sempre tem cards)
